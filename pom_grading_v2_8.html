POM 그레이딩 편차표 생성기

Base Size 기준 · 스펙 + 편차 2행 표시
엑셀/PDF에서 복사 → 붙여넣기 → 다운로드
사용법
  1. Tech Spec(PDF 또는 엑셀)에서 POM 표를 마우스로 드래그 → Ctrl+C 복사
  2. 아래 입력란에 Ctrl+V 붙여넣기
  3. Base Size 확인 후 "편차표 생성" 클릭
  4. 결과 확인 후 엑셀 다운로드
💡 P/S, TOL, 사이즈 헤더 행이 포함되어도 자동으로 처리됩니다. POM 코드(A300, A412H 등)가 있는 행만 변환됩니다.

2 — 결과 편차표
